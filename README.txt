Mohon perhatian sebelum menggunakan ini:

1. Mohon untuk pastikan sistem sudah mempunyai composer 
2. Mohon untuk menyesuaikan value pada app_baseURL (baris 25) dan konfigurasi database(baris 33-36)  pada file .env
3. Untuk menjalankan local server website ini, bapak/ibu dapat membuka terminal yang mengarah ke folder website ini dan ketik 'php spark serve' lalu enter.
4. Untuk mengakses halaman khusus USER, bapak/ibu dapat membuka link baseURLnya saja (contoh: http://localhost/) dengan akun testing (jika diperlukan) sbg berikut:
    email: user@gmail.com
    password: user12345
5. Untuk mengakses halaman khusus ADMIN, bapak/ibu dapat membuka link baseURLnya saja dan menambahkan '/adminPage' (contoh http://localhost:8081/adminPage) dengan akun testing (jika diperlukan) sbg berikut:
    email: admin@admin.com
    password: admin12345


Sekian dan terima kasih.
